# Numerical ultrathin OSFET model
Publicly available OSFET model from Stanford University. This model was presented at SISPAD 2024.
Usage [SGFET]: python3 osfet_numerical_model_v1.py | tee osfet_numerical_model_v1.log
Usage [DGFET]: python3 dgosfet_numerical_model_v1.py | tee dgosfet_numerical_model_v1.log
